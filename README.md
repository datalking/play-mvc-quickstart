# play-mvc-quickstart   
> demo for play-mvc   

## dev 
```sh
 git clone https://github.com/datalking/play-mvc-quickstart.git
 cd play-mvc-quickstart/
 mvn clean install
```

## demo
```sh
  cd play-mvc-quickstart/
  mvn tomcat7:run
```

start from [http://localhost:8999](http://localhost:8999)
